Create Ansible Playbook, to be run after the ansible playbooks of prepare_environment and install_docker

This snsible playbook will be run ONLY on master node for now.

Ansible playbook name:

install_k8s_kind_cluster.yaml

It should:

1- Install kind cluster as a docker container
2- Install Kubectl
3- Start cluster
4- Create service accounts, role binding, secret, and apply new CAs including the needed IP's
5- Retreive token and CA
6- Create the kubeconfig file with token, CA, and the server's public IP
7- Apply commands to disable unused components for lower resource consumption
